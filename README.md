# DXB VIP Lounge Service - FAQ Page

A Next.js application for the DXB VIP Lounge Service FAQ page with interactive accordion components.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Responsive FAQ accordion interface
- Modern black and gold design
- TypeScript support
- Tailwind CSS styling
- Jest testing setup

## Technologies Used

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Jest & Testing Library
